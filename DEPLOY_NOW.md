## üåê Option 1: GitHub Pages (Recommended - FREE)

### **Step 1: Enable GitHub Pages**

1. **Go to your repository**: [https://github.com/omarabdullatiff/my-portfolio-](https://github.com/omarabdullatiff/my-portfolio-)

2. **Click "Settings"** (top navigation bar)

3. **Click "Pages"** (left sidebar, under "Code and automation")

4. **Configure Source**:
   - **Source**: Select `Deploy from a branch`
   - **Branch**: Select `main`
   - **Folder**: Select `/ (root)`

5. **Click "Save"**

6. **Wait 2-5 minutes** for GitHub to build your site

7. **Your live URL**: `https://omarabdullatiff.github.io/my-portfolio-/`

### **Step 2: Verify Deployment**

- Check the "Pages" section - it should show "Your site is live at..."
- Visit your URL to see your portfolio live!

## üîß Troubleshooting

### **GitHub Pages not working?**
- Check repository is public (or you have GitHub Pro)
- Verify branch is `main`
- Wait 5-10 minutes for first deployment
- Check "Actions" tab for build errors

### **Site looks broken?**
- Clear browser cache
- Check file paths are correct
- Verify all files are committed
- Check browser console for errors
